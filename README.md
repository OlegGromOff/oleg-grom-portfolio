# Oleg Gromov - Frontend Developer Portfolio

Live Demo: [oleg-grom-portfolio.vercel.app](https://oleg-grom-portfolio.vercel.app)

## About The Project

This is my personal portfolio website built to showcase my projects, technical experience, and skills as a Frontend Developer based in Berlin. It is designed to be minimalistic, fast, and fully responsive.

## Tech Stack

- **Framework:** Next.js (App Router)
- **Library:** React
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Hosting:** Vercel

## Key Features

- 📱 Fully responsive design (Mobile-first approach)
- ⚡ Optimized performance and static rendering
- ⚖️ Fully GDPR Compliant (Includes Impressum & Privacy Policy pages as required by German law)
- 🎨 Modern dark-mode UI

## Getting Started

If you want to run this project locally, follow these steps:

1. Clone the repository:
   \`\`\`bash
   git clone https://github.com/YOUR_GITHUB_USERNAME/portfolio.git
   \`\`\`

2. Install dependencies:
   \`\`\`bash
   npm install
   \`\`\`

3. Run the development server:
   \`\`\`bash
   npm run dev
   \`\`\`

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
